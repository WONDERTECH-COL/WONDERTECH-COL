# Guía interna GitHub (GitHub Free) — WONDERTECH SAS

Documentación sencilla para operar repositorios de forma ordenada y empresarial.

---

## 1) Alcance

- Esta guía aplica **solo a repositorios creados después** de la fecha de entrada en vigencia de la política.

---

## 2) Nombre estándar de repositorios (obligatorio)

**Formato:**

- `[ID_ODOO]_NOMBRE_DEL_PROYECTO_CON_GUION_BAJO`

**Regla clave:**

- El **ID de Odoo SOLO va en el nombre del repositorio**, entre corchetes.
- **No** se repite en: README, carpetas, tags, releases, etc.

**Ejemplos:**

- `[523]_RUES`
- `[610]_INTEGRACION_WORDPRESS_ODOO`
